# Finanzas PRO

Dirigido a parejas que quieran gestionar sus finanzas juntos de una forma simple e intuitiva, sin dedicar horas a cuadrar números. Principalmente a novatos, o experimentados que quieran agilizar cuentas.

# MVP
- Pregunta número de participantes
- Toma datos a cada participante (nombre)

(INICIO MES) 
    - Pregunta a cada participante sus ingresos
    - Calcula monto total y lo muestra
    - Pregunta una configuración de aporte (50/50, proporcional al salario, personalizado) [También podría preguntarse antes]

    - Incluye una calculadora de gastos fijos, variables o deuda
    - Elegir % de estrategia, o $ 
    - Muestra como quedaría aplicando estrategia, permite al usuario aceptar o retroceder
    - Si acepta, muestra resumen para que el usuario tome nota si desea.

    - Tests para cada clase 
    - Persistencia básica
